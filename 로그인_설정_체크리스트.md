# 로그인(OAuth) 설정 체크리스트

## 1. 브라우저 주소

- 반드시 **http://localhost:3003** 으로 접속하세요.
- `127.0.0.1` 사용 시 콜백이 실패할 수 있습니다.

## 2. .env (next-app 폴더)

```
NEXTAUTH_URL=http://localhost:3003
```

- 끝에 `/` 없이, 접속 주소와 동일하게 설정
- 수정 후 **서버 재시작** (Ctrl+C 후 `npm run dev`)

## 3. OAuth 콘솔에 아래 URI를 그대로 등록

### Google

- **위치**: [Cloud Console](https://console.cloud.google.com/) → API 및 서비스 → 사용자 인증 정보 → OAuth 2.0 클라이언트 ID → 승인된 리디렉션 URI
- **등록할 URI**:
  ```
  http://localhost:3003/api/auth/callback/google
  ```

### 카카오

- **위치**: [카카오 개발자 콘솔](https://developers.kakao.com/) → 앱 → 로그인 → Redirect URI
- **등록할 URI**:
  ```
  http://localhost:3003/api/auth/callback/kakao
  ```

### 네이버

- **위치**: [네이버 개발자 센터](https://developers.naver.com/) → 애플리케이션 → API 설정 → Callback URL
- **등록할 URI**:
  ```
  http://localhost:3003/api/auth/callback/naver
  ```

---

## 운영(www.sejonglab.com) 배포 시

- `.env`의 `NEXTAUTH_URL`을 `https://www.sejonglab.com` 로 변경
- 각 OAuth 콘솔에 **추가**로 등록:
  - `https://www.sejonglab.com/api/auth/callback/google`
  - `https://www.sejonglab.com/api/auth/callback/kakao`
  - `https://www.sejonglab.com/api/auth/callback/naver`
